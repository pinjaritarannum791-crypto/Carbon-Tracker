# EcoSense – AI Carbon Footprint Tracker
*SDG 13 – Climate Action*

## Problem
People are unaware of their daily carbon footprint from transport, diet, electricity, and shopping.

## Solution
A web app that calculates your annual CO₂ emissions and provides AI-powered tips to reduce them.

## Features
- Carbon footprint calculator
- AI analysis using Claude API
- Breakdown by category
- Personalised reduction tips

## How to Run
Download and open carbon_footprint_tracker.html in any browser. No installation needed.

## Tech Stack
HTML, CSS, JavaScript, Claude AI (Generative AI / NLP)
